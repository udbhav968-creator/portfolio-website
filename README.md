## Features
- **Modern & Responsive Design:** Built with React and Vanilla CSS, featuring a sleek dark-mode aesthetic with glassmorphism effects.
- **Dynamic UI:** Smooth scroll animations, responsive navigation, and vibrant gradient accents.
- **Comprehensive Sections:**
  - **Hero:** Quick introduction and focus areas.
  - **About:** Academic details and categorized technical skills.
  - **Experience:** Timeline of internships and leadership roles.
  - **Projects:** Interactive cards showcasing major technical projects.
  - **Contact:** Form to get in touch directly.

## Tech Stack
- **Frontend:** React (Vite)
- **Styling:** Vanilla CSS
- **Icons:** React-Icons
- **Deployment:** Vercel

## Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/udbhav968-creator/portfolio-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173`.
